# Code of Conduct

## Notre engagement

Dans l'intérêt de favoriser un environnement ouvert et accueillant, nous nous engageons à faire de la participation à ce projet une expérience sans harcèlement pour tous.

## Nos standards

Comportements qui contribuent à un environnement positif :
- Utiliser un langage accueillant et inclusif
- Respecter les points de vue et expériences différents
- Accepter gracieusement les critiques constructives
- Se concentrer sur ce qui est le mieux pour la communauté

Comportements inacceptables :
- Langage ou images à caractère sexuel
- Trolling, commentaires insultants, attaques personnelles
- Harcèlement public ou privé
- Publier des informations privées d'autrui sans permission

## Application

Les comportements abusifs peuvent être signalés en ouvrant une issue ou en contactant directement le mainteneur. Toutes les plaintes seront examinées et traitées.

Ce Code of Conduct est adapté du [Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
